以下都是入门教程，具体见官方文档。

-------------------begin---
	BrowserSync
1,Enter the BrowserSync floder;
2,key-in 'node start.js';
3,then you'll see the browser auto-start
浏览器自动刷新
--------------------end---


--------------------begin---
	BrowserSync+gulp
1,Enter the BrowserSync+gulp floder;
2,key-in 'gulp watch';
3,you can modify the 'index.html' file in the root directory,then you'll find the browser auto refresh.
加了一个gulp而已..
--------------------end---


--------------------begin---
	BrowserSync+gulp-nodemon
1,Enter the BrowserSync+gulp-nodemon floder;
2,key-in 'gulp browserSync';
3,you can modify the '/views/index.html',then you'll find the browser auto refresh.
这个是当服务器那边(express)改动的时候浏览器自动刷新的
--------------------end---
